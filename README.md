# MemoryX-API
The MemoryX REST API

## Getting started
### Prerequisites
- [Node.js](https://nodejs.org/en/) (v10.15.3)
- [npm](https://www.npmjs.com/) (v6.4.1)

### Installation
1. Clone the repository
2. Install dependencies
```bash
npm install
```

### Initialize the database
```bash
npm run generate
```

### Run the server
```bash
npm run dev
```

## Run tests
```bash
npm run test
```
